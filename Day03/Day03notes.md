# SDLC Task 1

## Choose One App: WhatsApp

### Requirement
- Users should be able to send messages, make voice calls, and make video calls.

### Planning
- Decide the project timeline.
- Estimate the budget.
- Assign the development, testing, and design teams.
- Define project goals and resources.

### Design
- Design the WhatsApp logo.
- Create the UI/UX for all screens.
- Design the database structure and application architecture.

### Development
- Developers write the frontend and backend code.
- APIs and database are implemented.
- Features like messaging, calling, and status are developed.

### Testing
- QA engineers test every feature using different test scenarios.
- Identify and report bugs to developers.
- Verify bug fixes after developers resolve them.

### Deployment
- After successful testing, the application is released to users through the Play Store and App Store.
- New versions with tested features are deployed.

### Maintenance
- Fix bugs reported by users.
- Add new features and improvements.
- Improve security and application performance.
- Release regular updates.

## Sometimes interviewers ask tricky questions like:

- Bug: Company logo is incorrect on the login page one hour before launch.

Answer:

- Severity: Low (the application still works)
- Priority: High (it must be fixed before release because it affects the company's brand)
