# Day 1 Notes

## What is Software?
Software is a collection of programs, data, and instructions that tell a computer what to do.

## What is Software Testing?
Software testing is the process of checking whether a software application works as expected and identifying defects before it reaches users.

## What is QA?
Quality Assurance (QA) is a process that ensures software is developed and tested according to quality standards.

## Why is Testing Important?
- Finds bugs before users.
- Improves quality.
- Saves money.
- Builds customer trust.

## Responsibilities of a QA Engineer
- Understand requirements.
- Design test cases.
- Execute tests.
- Report bugs.
- Verify bug fixes.

## QA vs Developer

### Developer
- Writes code.
- Builds features.
- Fixes bugs.

### QA
- Tests software.
- Finds defects.
- Ensures quality.

## Verification vs Validation

Verification:
Are we building the product correctly?

Validation:
Are we building the correct product?
